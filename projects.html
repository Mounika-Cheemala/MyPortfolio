Restaurant Automation System

The *Restaurant Automation System* is designed to streamline restaurant operations by automating the ordering process, managing inventory, and handling billing efficiently. The system provides an intuitive interface for both staff and customers, reducing manual errors and improving service speed.

Technologies Used:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Java
  • Database: MySQL

My Role:

I was responsible for designing and developing the database for the system, ensuring efficient data storage and seamless integration with the backend.
